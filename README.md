# Otelmeister

Tool for manipulating and viewing OTEL and Jaeger dump files.


## Install

```bash
cargo install --locked -f otelmeister
```

## Demo

```console
$ fetch-my-trace 8908c5e22c986c3c | otelmeister view
```
